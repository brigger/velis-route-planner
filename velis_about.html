About this planner

This project started because I love flying the Pipistrel Velis Electro — and because planning an electric flight with the POH tables alone was cumbersome. Every leg meant flipping between ROC charts, %SOC-per-phase tables and cruise-power interpolation by hand.

So I built the tool I wanted to have in the cockpit: sliders for the three things that actually change (SOH, OAT, cruise power), dynamic legs, and a single %SOC number at the end. It's free, open, and still evolving.

— Patrick
